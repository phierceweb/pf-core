# Code of Conduct

## Our standard

pf-core is a small, focused project. Everyone interacting in its issues, pull
requests, and discussions is expected to be respectful and constructive.

**Expected:** thoughtful feedback, good-faith disagreement, patience with
newcomers, and keeping discussion technical.

**Not tolerated:** harassment, personal attacks, demeaning or discriminatory
comments, publishing others' private information, or sustained disruption.

## Scope

This applies in all project spaces — issues, pull requests, commit
discussions, and any other official channel — and when representing the project
in public spaces.

## Reporting and enforcement

Report unacceptable behavior privately to the project maintainer at
**oss@phierceweb.com**. Reports will be handled discreetly. Maintainers may
edit or remove contributions, and may warn or ban contributors, for behavior
that violates this code.

---

For a more comprehensive document, this project is happy to adopt the
[Contributor Covenant](https://www.contributor-covenant.org); the above is a
concise statement of the same principles.
